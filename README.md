# The Hunt - ユーザーマニュアル

FFXIV モブハント管理ツール「The Hunt」の使い方と注意点についての説明書です。

## 1. 画面の見方

### ヘッダー
- **ET (エオルゼア時間)**: 現在のETを表示します。
- **ランクタブ**: 表示するモブのランク (S, A, F, ALL) を切り替えます。
  - **クリック挙動  (同じタブをクリック)**:
    - **1回目**: そのランクに切り替えます。
    - **2回目**: エリアフィルタパネルを表示します。
    - **3回目**: エリアフィルタパネルを閉じます。
- **エリアフィルタ**: 拡張パッチごとのエリアで表示を絞り込めます。
  - **個別ボタン**: クリックで選択/解除を切り替えます（複数選択可）。
  - **全選択/全解除**:
    - 全て選択されている状態で押すと「全解除」になります。
    - それ以外の場合は「全選択」になります。
  - **ALLタブ時の挙動**: エリアではなく「ランク (S, A, F)」の絞り込みフィルタとして機能します。

### モブカード
各モブの状況を表示するカードです。
- **プログレスバー**: 湧き時間までの進行度を表示します。
  - **青/水色**: 通常進行中。
  - **カード枠 (白点滅)**: 特殊条件モブの湧き条件を満たしている、または湧き時間内 (Condition Active)。
  - **バー赤点滅 (MaxOver)**: 湧き時間の90%を超過しています。
- **右上のアイコン**:
  - 📝: メモが登録されています。カーソルを合わせると内容を表示します。
  - 📍: Sランクの湧き位置情報があります (クリックで展開)。

## 2. 操作方法

### 討伐報告
モブを討伐した際に報告を行います。
- **Aランク**: カード右上の「報告」ボタンを押すと、**即時**に現在時刻で報告されます。
- **S、Fランク**: 「報告」ボタンを押すと別窓が開きます。討伐時刻を指定して報告できます。
  - **注意**: 明らかに湧き時間より早い報告はエラーになります。時間修正する場合など左側のチェックを入れることで無視して送信可能です。

### Sランク詳細 (湧き位置・メモ)
Sランクのカードをクリックすると詳細パネルが展開します。
- **メモ**: 共有メモを入力・閲覧できます。
- **湧き位置マップ**: 候補地点が表示されます。
  - **湧き潰し**: 候補地点をクリック（タップ）することで「湧き潰し済（グレー）」と「未確認（色付き）」を切り替えられます。
  - **スマホでの操作**: 誤操作防止のため、**ダブルタップ（1秒以内に2回タップ）** で切り替わります。シングルタップではツールチップ（番号）が表示されます。

## 3. 注意事項

### メンテナンス時の挙動
- **データ取得**: メンテナンス情報は自動的に取得されます。
- **表示**:
  - **メンテナンス時間以降**: 次回の湧きがメンテナンスに被る場合、カードはグレーアウトしますが、**報告等の操作は可能**です。
  - **メンテナンス停止中**: カードがグレーアウトし、操作不可（報告ボタン無効）になります。
- **並び順**: メンテナンス影響下のモブは、リストの**一番下**に自動的に並び替えられます。

### データの同期
- 他のユーザーの報告や湧き潰し状況はリアルタイムで反映されます。
- 画面を開いたままにしておけば、自動的に最新の状態になります。

### 推奨環境
- **ブラウザ**: Google Chrome, Firefox, Safari, Edge の最新版。
- **JavaScript**: 有効にしてください。
